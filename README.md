# Capstone_2
## EmporiUm Sales Analysis Capstone Project

This repository contains my second capstone project analysis for EmporiUm, a fictional student bookstore chain that also sells tech products, art supplies, and other merchandise. The project focuses on comparing sales performance between the Northeast territories of Maine and Maryland. This repository includes a Jupyter Notebook with the full analysis, Python code, data visualizations, key findings, and business insights. It also contains a short presentation video explaining my approach, analysis process, and final conclusions. Through this project, I strengthened my skills in data cleaning, visualization, and data analysis using Python tools such as NumPy, Pandas, and Matplotlib.
